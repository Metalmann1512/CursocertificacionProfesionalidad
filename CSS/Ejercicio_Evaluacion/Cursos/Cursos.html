Niveles de los cursos

Principiante

Este nivel es para aquellos que nunca han practicado artes marciales antes. Aprenderás los fundamentos básicos y desarrollarás una base sólida.

Intermedio

Este nivel es para aquellos que conozcan los principios basicos y quieran aprender conceptos mas avanzados

Avanzado

Este nivel es para aquellos que tienen una base solida y potente. En este nivel aprenderas realmente como pelea un dragón

Beneficios de practicar artes marciales

  • Mejora la condición física
  • Aumenta la confianza en uno mismo
  • Desarrolla habilidades de defensa personal
  • Fomenta la disciplina y el autocontrol
  • Promueve la socialización y el trabajo en equipo
