Preguntas Frecuentes

¿Qué tipo de cursos ofrecen?

Ofrecemos una variedad de cursos de artes marciales, incluyendo Karate, Taekwondo y Jiu-Jitsu.

¿Cuáles son los requisitos para inscribirse?

No se requieren requisitos previos. Todos los niveles son bienvenidos.

¿Qué ropa debo llevar a la primera clase

Recomendamos llevar ropa cómoda y deportiva. Si tienes un uniforme de artes marciales, puedes usarlo.

¿Hay clases para niños?

Sí, ofrecemos clases para niños a partir de 5 años.

¿Puedo asistir a una clase de prueba?

Sí, ofrecemos una clase de prueba gratuita para que puedas probar antes de inscribirte.
